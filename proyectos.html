Proyectos Destacados

Estos son mis proyectos más relevantes como administrador de sistemas y redes con especialidad en ciberseguridad

Scripts de Seguridad y Automatización Multiplataforma

Implementación de scripts en Windows, Linux y macOS para tareas críticas de mantenimiento, respaldo, auditoría e integridad del sistema. Diseñados para entornos reales y con enfoque en buenas prácticas de ciberseguridad. Ideal para administradores que buscan automatizar tareas de seguridad sin depender de herramientas propietarias.

Sistema de Monitorización y Análisis de Tráfico de Red

Entorno de monitorización basado en NetFlow/sFlow para recopilar, analizar y visualizar tráfico de red en tiempo real. Incluye configuración de dispositivos de red, servidor de recolección y detección de anomalías como escaneos de puertos o picos de tráfico. Permite identificar fugas de datos, ataques DDoS o uso no autorizado del ancho de banda. Solución escalable y apta para entornos educativos y PYMEs.

Firewall Personalizado con iptables

Implementación de reglas avanzadas de filtrado de tráfico en Linux usando iptables. Incluye protección contra escaneos de puertos, bloqueo de IPs maliciosas y logging de intentos de acceso no autorizados.

Análisis Forense de Tráfico de Red

Captura y análisis de paquetes con Wireshark para identificar protocolos inseguros, comunicaciones sospechosas y fugas de información. Incluye detección de tráfico HTTP sin cifrar y patrones de ataque comunes.

Auditoría de Contraseñas con John the Ripper

Pruebas de fuerza bruta y diccionario sobre hashes de contraseñas extraídos de sistemas Linux y Windows. Se evalúa la fortaleza de las políticas de contraseñas y se recomiendan mejoras según estándares NIST.

Sistema de Monitoreo de Integridad de Archivos

Uso de herramientas como AIDE (Advanced Intrusion Detection Environment) para detectar modificaciones no autorizadas en archivos críticos del sistema, como /etc/passwd o binarios del sistema.

Endurecimiento del Servicio SSH

Configuración segura del servidor SSH: desactivación de root login, uso de claves públicas, cambio de puerto, limitación de intentos y registro de accesos. Ideal para servidores expuestos a Internet.

Detección de Malware en Sistemas Linux

Uso de herramientas como ClamAV y chkrootkit para identificar troyanos, rootkits y scripts maliciosos en servidores. Incluye automatización del escaneo y generación de informes diarios.

Pruebas de Seguridad en Redes WiFi

Escaneo de redes inalámbricas con Aircrack-ng, análisis de cifrados (WEP/WPA/WPA2) y demostración de vulnerabilidades en configuraciones débiles. Todo en entorno controlado y ético.

Centralización y Análisis de Logs del Sistema

Configuración de rsyslog para enviar logs a un servidor central, y uso de grep/sed/awk para detectar patrones de intrusión, como múltiples fallos de autenticación o accesos fuera de horario.

Cifrado de Archivos y Discos con LUKS

Protección de datos sensibles mediante cifrado de particiones con LUKS en Linux y uso de GPG para cifrar archivos individuales. Incluye gestión segura de claves y recuperación de emergencia.

Políticas de Seguridad en Windows con GPO

Configuración de Group Policy Objects (GPO) para restringir USB, forzar actualizaciones, deshabilitar PowerShell innecesario y aplicar contraseñas complejas en entornos empresariales.

Honeypot Básico para Detección de Ataques

Despliegue de un honeypot simple con Docker o Python para atraer y registrar intentos de conexión maliciosa. Útil para entender tácticas de atacantes y mejorar defensas perimetrales.

Escaneo y Mapeo de Red con Nmap

Identificación de hosts activos, servicios expuestos y versiones de software en una red local. Incluye scripts NSE para detección de vulnerabilidades conocidas y generación de reportes.

Configuración Segura de un Servidor DNS

Instalación y endurecimiento de BIND9: desactivación de transferencias de zona, uso de DNSSEC básico, logging de consultas y protección contra ataques de amplificación.

Backup Automatizado y Cifrado

Script que realiza copias de seguridad incrementales, las cifra con GPG y las sube a un repositorio remoto seguro. Incluye verificación de integridad y rotación automática de backups.

Introducción Ética a Metasploit Framework

Uso controlado de Metasploit para explotar vulnerabilidades en máquinas virtuales de prueba (como Metasploitable). Enfoque en comprensión de exploits, payloads y post-explotación segura.

Escaneo de Vulnerabilidades Web con Nikto

Análisis automatizado de servidores web para detectar configuraciones inseguras, directorios expuestos, versiones antiguas de Apache/Nginx y riesgos comunes de OWASP Top 10.

Gestión Segura de Usuarios en Linux

Automatización de creación de usuarios con políticas de contraseñas, asignación a grupos específicos, expiración de cuentas y auditoría periódica con scripts en Bash.

Plan Básico de Respuesta a Incidentes

Documento y script de respuesta ante incidentes: aislamiento de host, recolección forense (logs, procesos, conexiones), notificación y restauración desde backup limpio.
