Proyectos Destacados

Estos son mis proyectos más relevantes como administrador de sistemas y redes con especialidad en ciberseguridad

Scripts de Seguridad y Automatización Multiplataforma

Implementación de scripts en Windows, Linux y macOS para tareas críticas de mantenimiento, respaldo, auditoría e integridad del sistema. Diseñados para entornos reales y con enfoque en buenas prácticas de ciberseguridad. Ideal para administradores que buscan automatizar tareas de seguridad sin depender de herramientas propietarias.

Sistema de Monitorización y Análisis de Tráfico de Red

Entorno de monitorización basado en NetFlow/sFlow para recopilar, analizar y visualizar tráfico de red en tiempo real. Incluye configuración de dispositivos de red, servidor de recolección y detección de anomalías como escaneos de puertos o picos de tráfico. Permite identificar fugas de datos, ataques DDoS o uso no autorizado del ancho de banda. Solución escalable y apta para entornos educativos y PYMEs.

Script de Auditoría Básica de Usuarios en Linux

Script en Bash que lista usuarios con UID 0, verifica cuentas sin contraseña y muestra últimas conexiones. Ideal para comprobar configuraciones seguras tras instalar un sistema operativo.

Bloqueo Automático de IPs con Múltiples Fallos de Login

Script que analiza /var/log/auth.log y añade reglas temporales en iptables para bloquear IPs con más de 3 intentos fallidos de acceso SSH.

Configuración Segura de un Servidor Web Apache

Endurecimiento básico: ocultar versión del servidor, deshabilitar listado de directorios, restringir métodos HTTP y configurar cabeceras de seguridad como X-Content-Type-Options.

Análisis de Tráfico: HTTP vs HTTPS con Wireshark

Captura y comparación de tráfico web cifrado y no cifrado. Se demuestra cómo las credenciales viajan en texto claro sin TLS, reforzando la importancia del cifrado.

Política de Contraseñas en Windows con Editor de Directivas Locales

Configuración de longitud mínima, complejidad, historial y expiración de contraseñas en Windows 10/11 mediante secpol.msc, aplicable en equipos independientes.

Copia de Seguridad Automatizada con Robocopy

Script en PowerShell que sincroniza carpetas críticas con una unidad externa usando Robocopy, programado con el Programador de Tareas de Windows.

Escaneo de Puertos con Nmap en Red Local

Identificación de dispositivos activos y servicios expuestos en una red doméstica usando comandos básicos de Nmap, sin scripts ni explotación.

Cifrado de Archivos Sensibles con GPG

Uso de GnuPG para cifrar documentos confidenciales mediante clave simétrica, ideal para proteger información en unidades USB o transferencias internas.

Monitorización de Uso de Disco con Alertas

Script en Bash que comprueba el uso del disco con df y genera un archivo de alerta si supera el 85%, base para sistemas de monitoreo sencillos.

Reglas Básicas de Firewall en Windows Defender

Creación de reglas entrantes/salientes para permitir o bloquear aplicaciones específicas (como un servidor web local) usando el Panel de Control de Windows.

Registro de Dispositivos USB Conectados en Linux

Script que consulta dmesg y /var/log/syslog para listar dispositivos USB conectados recientemente, útil en auditorías de hardware en laboratorios.

Verificación de Integridad con Funciones Hash

Uso de sha256sum (Linux) y Get-FileHash (PowerShell) para comprobar que un archivo descargado no ha sido modificado, aplicado a ISOs de sistemas operativos.

Servidor FTP Básico con Autenticación y Chroot

Instalación y configuración de vsftpd en Linux con usuarios locales, directorio raíz restringido y sin acceso anónimo, siguiendo buenas prácticas mínimas.

Análisis Básico de Logs del Sistema

Uso de grep, tail y less para buscar errores, accesos fallidos o reinicios en archivos como /var/log/syslog o /var/log/auth.log en Linux.

Automatización de Limpieza de Archivos Temporales

Scripts en Bash y PowerShell que eliminan archivos antiguos en /tmp o %TEMP%, mejorando rendimiento y reduciendo riesgos de acumulación de datos sensibles.

Configuración Manual de Servidores DNS

Modificación de resolv.conf en Linux y ajustes de adaptador en Windows para usar servidores DNS públicos seguros como Cloudflare (1.1.1.1).

Automatización de Creación de Usuarios en Linux

Script en Bash que lee un archivo CSV con nombres y crea usuarios con directorio home, grupo asignado y contraseña temporal, útil en entornos educativos.

Diagnóstico Básico de Problemas de Red

Uso combinado de ping, traceroute, nslookup y ipconfig/ifconfig para identificar fallos de conectividad, resolución DNS o rutas incorrectas en una red local.
